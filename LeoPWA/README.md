# L'eco di Leopardi

PWA didattica installabile dedicata al pensiero di Giacomo Leopardi.

## Contenuto
- lezioni testuali
- video incorporati
- schemi e mappe concettuali
- glossario interattivo
- quiz di autoverifica
- appunti personali esportabili in `.txt`

## Pubblicazione su GitHub Pages
1. carica tutti i file nella root del repository
2. attiva GitHub Pages dalla branch principale
3. attendi la pubblicazione
4. apri `index.html` e installa la PWA dal browser, se supportato
