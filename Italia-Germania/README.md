# Italia 1919–1939 — PWA

## Avvio in locale
Per testare Service Worker/PWA serve un server (non aprire i file con doppio click).

```bash
python3 -m http.server 8000
```

Apri: http://localhost:8000

## GitHub Pages
Carica il contenuto della cartella (root) in un repository e attiva Pages su branch `main` / root.

## Installazione
- Android (Chrome): menu ⋮ → “Installa app”
- iOS/iPadOS (Safari): Condividi → “Aggiungi a Home”

## Video / Audio
Il toggle “Audio: ON/OFF” usa YouTube IFrame API.
Su iOS spesso l’audio parte disattivato: tocca “Audio: ON”.
